USA States

Each page includes the following:
State Abbreviation: The two-letter abbreviation for each state.
Statehood: When each state was admitted into the Union.
State Capital: The capital city of the state.
State Size: The area of the state, both total (land + water) and land only.
Number of Counties: The number of counties for each state.
State Electoral Votes: The number of electoral votes each state has.
State Motto: The motto for each state.
State Flower: The state flower as designated by each state.
State Bird: The state bird as designated by each state.
State Nickname: The nickname for each state. State Flag: The picture for the state flag for each state.
Area Codes: The telephone area codes for each state.
Top 5 Cities: The top 5 cities for each state.
Major Sports Teams: The major teams for each state.
